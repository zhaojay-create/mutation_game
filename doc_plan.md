# Mutation 游戏策划文档

## 游戏概述

- **游戏名**：Mutation
- **类型**：塔防 + 半自由放置 + 变异育成
- **核心循环**：放置单位 → 利用土壤变异强化单位 → 抵御四方涌来的敌人 → 保护地图中央核心

---

## 核心目标

- 地图中央有一个**绿色核心**，是玩家需要保护的对象
- 敌人的最终目标是摧毁绿色核心
- 核心血量归零 → **游戏失败**
- 玩家通过放置并变异各类单位来阻挡/消灭敌人

---

## 放置单位（Placeable Units）

| 类型     | 说明                                                                         |
| -------- | ---------------------------------------------------------------------------- |
| **植物** | 固定防御单位，被动攻击或产出资源；受土壤影响变异出不同攻击类型               |
| **动物** | 主动攻击单位，可对范围内敌人发动进攻；受土壤影响获得不同战斗属性             |
| **蛋**   | 放置后经过孵化时间变成特殊角色（如：龙）；土壤类型决定孵化出的角色种类和属性 |

### 蛋的孵化示例

- 普通土 → 普通龙
- 火焰土 → 火龙（攻击带灼烧）
- 毒土 → 毒龙（攻击附带中毒）
- 水土 → 冰龙（攻击减速）

---

## 半自由放置系统

### 操作特性

- **鼠标自由移动**：拖动鼠标在地图任意位置预览放置
- **隐藏吸附网格**：表现上自由移动，底层坐标吸附到最近的隐藏格点，保证布局整齐
- **最小占地面积**：每个单位有碰撞体积，定义最小占地范围
- **碰撞检测**：单位之间不可重叠，放置时检测与已有单位的碰撞
- **地形阻挡**：障碍物、水域等地形不可放置
- **红/绿预览**：
  - 🟢 绿色预览 → 当前位置可放置
  - 🔴 红色预览 → 当前位置不可放置（碰撞/地形阻挡）

### 动态放置

- **波次进行中仍可放置和变异单位**，无需暂停
- 变异效果**即时生效**，不存在生长/延迟阶段

---

## 变异系统（Mutation System）

### 触发逻辑

单位放置后，所在位置的**土壤类型**决定该单位的变异方向。变异即时生效。

### 土壤类型

| 土壤           | 变异效果方向                         |
| -------------- | ------------------------------------ |
| 普通土         | 无变异，保持基础属性                 |
| 火焰土         | 攻击附带灼烧，单位获得火焰外观       |
| 毒土           | 攻击附带中毒，持续伤害               |
| 水土           | 攻击减速敌人，或提供治疗光环         |
| 暗土           | 单位隐身（敌人不优先攻击）或暴击爆发 |
| （更多待扩展） | —                                    |

### 土壤来源

1. **地图预设**：关卡中预先布置好不同类型的土壤区域，玩家可利用地形规划放置策略
2. **玩家主动修改**：玩家可消耗资源在指定区域改变土壤类型，主动引导变异

### 变异效果示例

- 荆棘植物 + 火焰土 → **焚烧荆棘**，攻击带灼烧效果
- 动物 + 毒土 → **毒爪野兽**，爪击附带中毒
- 蛋 + 水土 → 孵化出**冰霜龙**，攻击减速敌人

---

## 敌人系统

### 基础行为

- 从地图**四个方向**（或随机刷新点）持续生成，按波次入场
- 拥有**寻路 AI**，最终目标是到达并攻击**绿色核心**
- **优先攻击路径上的变异放置单位**，突破防线后才直奔核心

### 敌人类型（规划）

| 类型       | 特点                         |
| ---------- | ---------------------------- |
| 基础近战型 | 普通血量，普通速度，正面强攻 |
| 远程轰炸型 | 在路径中段停下远程攻击单位   |
| 高速冲锋型 | 速度极快，尝试绕过防线       |
| 精英/Boss  | 高血量高伤害，出现在特定波次 |

### 寻路逻辑

- 使用寻路算法（如 A\*）动态计算到核心的路径
- 遇到放置单位会优先攻击，击破后继续寻路
- 地形障碍会引导敌人绕行

---

## 核心（Core）

- 位于**地图中央**，固定不可移动
- 显示为**绿色**标识，视觉上清晰可辨
- 具有血量值，受到攻击后血量降低
- 血量归零 → **游戏失败**
- 核心自身不攻击敌人，纯防御目标

---

## 游戏阶段（单关卡原型）

```
准备阶段
  └─ 玩家放置初始单位，规划土壤区域
      ↓
波次开始
  └─ 敌人按波次从四方涌入
      ↓
动态防御
  └─ 波次中途仍可放置单位 / 修改土壤 / 触发变异（即时生效）
      ↓
波次结算
  └─ 核心存活 → 继续下一波
  └─ 核心摧毁 → 游戏失败
```

## 待扩展 / 后续讨论

- **资源系统**：放置单位和修改土壤需要消耗什么资源？如何获取？
- **单位升级路径**：变异之外是否有独立的升级树？
- **更多土壤类型**：雷、冰、圣、腐败等
- **变异组合连锁**：特定变异组合触发额外效果
- **游戏模式**：无限波次生存 / 多关卡 / Roguelite 永久成长
- **音效与视觉反馈**：变异特效、敌人受击反馈等
